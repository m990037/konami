# こなみへの愛のランディングページ 💕

亜紀裕からこなみへ、愛を伝えるための特別なウェブサイトです。

## 🎨 特徴

- **温かみのあるカジュアルなデザイン**: 柔らかい色合いとグラデーション
- **レスポンシブ対応**: スマートフォン、タブレット、PCで美しく表示
- **アニメーション効果**: スクロールに応じた滑らかなアニメーション
- **思い出のタイムライン**: 二人の物語を時系列で表示
- **写真ギャラリー**: 大切な思い出の写真を追加可能
- **特別なメッセージ**: 心からの愛のメッセージ

## 📂 ファイル構成

```
konami/
├── index.html      # メインHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScriptアニメーション
└── README.md       # このファイル
```

## 🚀 使い方

1. `index.html` をブラウザで開く
2. または、ローカルサーバーを起動:
   ```bash
   # Python 3の場合
   python -m http.server 8000
   
   # Node.jsの場合
   npx serve
   ```
3. ブラウザで `http://localhost:8000` にアクセス

## ✏️ カスタマイズ方法

### 記念日の日付を変更する

`index.html` の記念日セクション（約100行目付近）を編集:

```html
<p class="date">2023年4月15日</p>  <!-- 実際の日付に変更 -->
```

### 写真を追加する

1. プロジェクトフォルダに `images` フォルダを作成
2. 写真を `images` フォルダに保存
3. `index.html` の写真プレースホルダーを編集:

```html
<div class="gallery-item">
    <img src="images/photo1.jpg" alt="思い出の写真">
</div>
```

4. `styles.css` に画像スタイルを追加:

```css
.gallery-item img {
    width: 100%;
    height: 100%;
    object-fit: cover;
}
```

### タイムラインの内容を変更する

`index.html` のタイムラインセクション（約40行目付近）を編集:

```html
<div class="timeline-item">
    <div class="timeline-dot"></div>
    <div class="timeline-content">
        <h3>あなたのタイトル</h3>
        <p>あなたのエピソード</p>
    </div>
</div>
```

### 色を変更する

`styles.css` の色変数を編集:

```css
/* メインカラー: #d63447 (赤系) */
/* グラデーション: #ff9a9e, #fecfef, #ffecd2 (ピンク〜オレンジ系) */
```
